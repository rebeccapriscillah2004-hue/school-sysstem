# School System Project

## Overview
The School System is designed to facilitate the management of various school-related activities. It provides a platform for handling student information, managing courses, scheduling classes, and maintaining records efficiently.

## Features
- **Student Management**: Add, update, and delete student records.
- **Course Management**: Create and manage courses offered in the school.
- **Class Scheduling**: Schedule classes and allocate classrooms.
- **Report Generation**: Generate various reports for administrative needs.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rebeccapriscillah2004-hue/school-sysstem.git
   ```
2. Navigate into the project directory:
   ```bash
   cd school-sysstem
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
Access the application at `http://localhost:3000` after starting the server.
